# paez-post1-u1

Laboratorio 1 - Configuración del entorno de desarrollo web

## Descripción

Este proyecto corresponde al Laboratorio 1 de Programación Web.  
Se configuró el entorno con VS Code, Git y GitHub, y se construyó una página HTML básica con estilos y scripts.

## Instrucciones de ejecución

1. Clonar el repositorio.
2. Abrir la carpeta en VS Code.
3. Ejecutar `index.html` con Live Server.
4. Inspeccionar la página con Chrome DevTools.

## Capturas de Checkpoints

- Instalación de extensiones en VS Code.
- Configuración de Git.
- Repositorio clonado.
- Página HTML renderizada con estilos.
- Historial de commits en GitHub.
